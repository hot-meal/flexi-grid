Flexi Grid

A simple flexbox grid system.

Made by @hot-meal

Light weight

The Flexi Grid system is very lightweight, at only ~7kb minified using default settings.

Responsive

Classes for medium and small devices, the Flexi Grid can easily make your layouts responsive.

Customisable

The Flexi Grid system is written entirely in sass, so you can easily modify or customize the grid to suit your needs.
